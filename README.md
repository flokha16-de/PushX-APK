
 PushX 1.7 – Android App

Welcome to PushX 1.7, a lightweight Android app built with JStudio and optimized for Android 8 and above.

This repository contains the APK file and a full Wiki with documentation.

---

 Features

- Clean and simple home screen  
- Aptoide account: aptoideuploader98  
- Optimized for older Android devices  
- Very small APK size  
- Offline-friendly  
- Built with JStudio  
- No ads, no trackers  

---

 Download

You can download the APK in this Repo.

 How to Install

1. Download the APK  
2. Open it on your device  
3. Allow “Unknown Sources” if required  
4. Install  
5. Launch PushX
